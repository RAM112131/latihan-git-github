Belajar Git Remote

Git Remote Itu Adalah ..

Perintah git remote adalah bagian dari Git yang digunakan untuk mengelola koneksi ke repositori lain. Koneksi ini bisa ke repositori milik sendiri di platform lain (seperti GitHub) atau repositori milik orang lain. Dengan `remote`, Anda bisa mendorong (push) perubahan Anda dan menarik (pull) perubahan dari orang lain.

git remote -v
Menampilkan daftar remote yang sudah terkonfigurasi di repositori Anda beserta URL-nya. Opsi `-v` (verbose) memastikan URL-nya ikut ditampilkan.
git remote add [nama] [url]
Menambahkan koneksi baru ke sebuah repositori remote. Beri nama yang singkat (misalnya, `origin` atau `upstream`) dan sertakan URL dari repositori tersebut.
git remote show [nama]
Menampilkan informasi detail tentang sebuah remote, termasuk URL, branch yang dilacak (tracked), dan status branch lokal terhadap branch di remote.
git remote rename [nama-lama] [nama-baru]
Mengganti nama sebuah koneksi remote dari `nama-lama` menjadi `nama-baru`.
git remote remove [nama]
Menghapus koneksi ke sebuah remote. Ini tidak akan menghapus repositori remote itu sendiri, hanya koneksi dari repositori lokal Anda.
